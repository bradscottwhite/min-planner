# Min Planner

App is running at https://bradscottwhite.github.io/min-planner/

###### Requirements:
- NPM (Node Package Manager)
- React
- TypeScript
- Tailwind.css
- Framer Motion
- AWS Amplify CLI

###### Usage:
Run `npm start` to start app on port 3030
